# Governance Glossary

Concise definitions for common governance terms. For process details, see the linked source documents.

| Term | Definition |
| --- | --- |
| **Motion Seed** | A pre-admission submission of intent used to populate Queue 1 (New Motions) in the agenda system. Motion Seeds originate from Dukes/Earls (via Duke sponsorship) and must be sponsored by the appropriate King(s) before they become a valid Motion; the admission gate then validates the motion record. See [The Legislative Branch](./legislative-branch.md) for sponsorship rules and [Conclave Agenda Control](./conclave-agenda.md) for queue handling. |
| **Motion** | A formal Legislative Branch declaration of intent that states **WHAT** the system proposes to do (never HOW).
